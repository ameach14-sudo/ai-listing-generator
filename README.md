# AI Listing Generator

An AI-powered tool for real estate agents to generate polished MLS listing descriptions in seconds.

## What it does

Paste in your property details — beds, baths, square footage, key features, neighborhood — and get a professional listing description ready to copy and use.

## Project Structure

```
ai-listing-generator/
├── frontend/       # UI (form + output)
├── backend/        # API server (Claude API integration)
└── docs/           # Notes, copy, pricing
```

## Stack (planned)

- Frontend: HTML/CSS/JS (simple, fast to ship)
- Backend: Node.js + Express
- AI: Claude API (claude-haiku-4-5 for cost efficiency)
- Hosting: TBD
